## 用户需求

修复边界拖拽时上下界时间反转的问题。

## 问题分析

- **resize-start**：向下拖会缩短时长，但当前代码限制的是向上拖的距离（错误）
- **resize-end**：向上拖会缩短时长，但当前代码限制的是向下拖的距离（错误）

## 核心功能

- 修正 resize-start 的边界限制：限制向下拖的最大距离
- 修正 resize-end 的边界限制：限制向上拖的最大距离

## 修改文件

`src/components/task/TaskSlider.vue`

## 修改内容

### onMove2D 回调中

**resize-start（第100行）：**

```typescript
// 当前（错误）
const clampedOffsetY = Math.max(offset.y, -maxDeltaPixels)
// 修正后
const clampedOffsetY = Math.min(offset.y, maxDeltaPixels)
```

**resize-end（第111行）：**

```typescript
// 当前（错误）
const clampedOffsetY = Math.min(offset.y, maxDeltaPixels)
// 修正后
const clampedOffsetY = Math.max(offset.y, -maxDeltaPixels)
```

### onEnd2D 回调中

**resize-start（第171行）：**

```typescript
// 当前（错误）
const clampedDeltaMinutes = Math.max(deltaMinutes, -maxDeltaMinutes)
// 修正后
const clampedDeltaMinutes = Math.min(deltaMinutes, maxDeltaMinutes)
```

**resize-end（第182行）：**

```typescript
// 当前（错误）
const clampedDeltaMinutes = Math.min(deltaMinutes, maxDeltaMinutes)
// 修正后
const clampedDeltaMinutes = Math.max(deltaMinutes, -maxDeltaMinutes)
```
