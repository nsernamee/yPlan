## 产品概述

为 yPlan 日程管理应用添加苹果风格的动态效果，提升视觉体验和交互品质。

## 核心功能

- **磨砂玻璃效果**：任务面板、顶部导航、底部导航栏、任务滑块全部采用半透明毛玻璃背景
- **按钮动态效果**：悬停放大、点击缩小、弹性动画、微光效果
- **面板弹出动画**：弹性入场、遮罩渐变、流畅过渡
- **拖拽效果增强**：阴影加深、磨砂增强、弹性释放
- **深色模式支持**：完整的深色模式磨砂和动效适配

## 技术栈

- **CSS 特性**：`backdrop-filter: blur()`、CSS 变量、`@keyframes` 动画
- **Tailwind CSS**：扩展主题配置，自定义工具类
- **Vue 3**：动态类绑定、过渡组件

## 实现方案

### 1. 磨砂玻璃实现

使用 `backdrop-filter` 实现原生磨砂效果：

```
浅色模式：
- 面板：blur(20px) + rgba(255,255,255,0.72)
- 导航：blur(16px) + rgba(255,255,255,0.85)
- 滑块：blur(8px) + 半透明任务色

深色模式：
- 面板：blur(20px) + rgba(30,30,30,0.85)
- 导航：blur(16px) + rgba(30,30,30,0.9)
- 滑块：blur(8px) + 降低饱和度的任务色
```

### 2. 按钮动效实现

```
悬停：transform: scale(1.02) + box-shadow 扩散
点击：transform: scale(0.96) + transition: 150ms
过渡：cubic-bezier(0.34, 1.56, 0.64, 1) 弹性曲线
```

### 3. 性能优化

- 使用 `will-change` 提示浏览器优化
- 磨砂效果仅在需要时启用
- 动画使用 `transform` 和 `opacity`，避免触发重排

## 设计风格

采用 Apple Human Interface Guidelines 设计语言，强调层次感、通透感和流畅性。

## 核心视觉特征

- **磨砂玻璃**：半透明背景 + 背景模糊，营造层次感
- **弹性动画**：使用弹性曲线，模拟物理反馈
- **微妙阴影**：多层阴影系统，增加立体感
- **流畅过渡**：200ms 统一过渡时间，自然舒适
