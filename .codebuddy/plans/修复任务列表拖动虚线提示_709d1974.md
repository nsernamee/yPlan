## 用户需求

修复从任务列表拖动任务到日历视图时的两个问题：

1. 拖动时没有虚线提示（DropIndicator 不显示）
2. 无法正确完成拖放操作

## 问题根因

从任务列表拖动时，`TaskListDrawer` 只更新了鼠标位置 `currentPosition`，但没有设置目标时间 `targetTime`。而 `DropIndicator` 的显示条件是 `dragStore.isDragging && dragStore.targetTime`，导致虚线提示不显示。

## 修复目标

- 在 dragover 事件中实时计算目标时间
- 显示虚线提示和时间气泡
- 确保 drop 操作正确创建日程

## 技术方案

### 修改文件

**1. src/components/calendar/DayView.vue**

- 增强 `handleDragOver`：根据鼠标 Y 坐标计算目标时间，调用 `dragStore.setTargetTime()`
- 在 `handleDrop` 结束时清除 `targetTime`
- 添加 `handleDragLeave` 清除状态

**2. src/components/calendar/WeekView.vue**

- 添加 `handleDragOver` 函数：计算目标时间和目标日期列
- 添加 `handleDrop` 函数：处理从任务列表拖放创建日程
- 在每列添加 dragover/drop/dragleave 事件绑定

### 关键逻辑

```typescript
// handleDragOver 核心逻辑
const rect = timeAxisRef.value.getBoundingClientRect()
const y = event.clientY - rect.top + timeAxisRef.value.scrollTop - HEADER_HEIGHT
const hour = Math.max(0, Math.min(23, Math.floor(y / HOUR_HEIGHT)))
dragStore.setTargetTime(`${hour.toString().padStart(2, '0')}:00`)
```
